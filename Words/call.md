## 词义\核心意境：

==任何建立联系的方式==
用声音（或现代技术）与某人/某物建立联系
大声呼唤 -> 召唤、召集(call together) -> 给……取名 -> 短暂拜访 -> 某事在召唤行动(call for) -> 做出判断/裁决(call a game) -> 打电话
1. 大声呼叫
2. 召唤，传唤，点名，召开
3. 给 ... 取名
4. 短暂拜访，停靠
5. 打电话
